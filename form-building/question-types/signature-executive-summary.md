# CMX1 Activity Studio — Signature / Executive Summary Question Types

> **Parent:** [`cmx1/form-building/shared-concepts.md`](../shared-concepts.md)

These are special-purpose question types not shown in the standard question type grid but available via the Question Type dropdown selector in the right panel.

- **Signature**: Captures a digital signature from the end-user
- **Executive Summary**: Provides a summary/overview section within the form

For universal concepts (answer table, evaluation settings, observations, recommendations, logic rules, footer controls), see **[shared-concepts.md](../shared-concepts.md)**.
