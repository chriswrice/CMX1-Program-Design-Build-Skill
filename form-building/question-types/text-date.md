# CMX1 Activity Studio — Text / Date / DateTime Question Types

> **Parent:** [`cmx1/form-building/shared-concepts.md`](../shared-concepts.md)

These question types capture free-text input (Text) or date/time selections (Date, DateTime). They are typically informational/data-capture questions rather than scored compliance questions.

For universal concepts (answer table, evaluation settings, observations, recommendations, logic rules, footer controls), see **[shared-concepts.md](../shared-concepts.md)**.
