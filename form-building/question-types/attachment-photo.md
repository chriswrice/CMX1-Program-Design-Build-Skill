# CMX1 Activity Studio — Attachment / Photo Question Types

> **Parent:** [`cmx1/form-building/shared-concepts.md`](../shared-concepts.md)

These are standalone question types that make attachment/photo capture the primary purpose of the question (as opposed to the footer photo/attachment controls which are supplementary to other question types).

For universal concepts (answer table, evaluation settings, observations, recommendations, logic rules, footer controls), see **[shared-concepts.md](../shared-concepts.md)**.
