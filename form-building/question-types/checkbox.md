# CMX1 Activity Studio — Checkbox Question Type

> **Parent:** [`cmx1/form-building/shared-concepts.md`](../shared-concepts.md)

The Checkbox question type allows multiple answer selections (multi-select). Unlike Yes/No or Button Group where only one answer is selected, checkboxes let end-users select any combination of options.

For universal concepts (answer table, evaluation settings, observations, recommendations, logic rules, footer controls), see **[shared-concepts.md](../shared-concepts.md)**.
